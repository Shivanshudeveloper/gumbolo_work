Parallax

Around supports 2 types of parallax effect: on page scrolling and cursor position.

Around component

In your document make sure to link to:

  • vendor/jarallax/dist/jarallax.min.js - for background image parallax
  • vendor/jarallax/dist/jarallax-element.min.js - for element parallax
  • vendor/parallax-js/dist/parallax.min.js - for cursor position parallax

Use this page as a reference.

<!-- Element parallax -->
<div class="d-flex">
  <div data-jarallax-element="40">
    <img src="path-to-image" alt="Alt text">
  </div>
  <div data-jarallax-element="-40">
    <img src="path-to-image" alt="Alt text">
  </div>
</div>
// Element parallax
.d-flex
  div(data-jarallax-element="40")
    img(src="path-to-image", alt="Alt text")
  div(data-jarallax-element="-40")
    img(src="path-to-image", alt="Alt text")

<!-- Cursor position parallax -->
<div class="parallax">
  <div class="parallax-layer position-relative" data-depth="0.1">
     <img src="path-to-image" alt="Layer">
  </div>
  <div class="parallax-layer position-relative" data-depth="0.35">
     <img src="path-to-image" alt="Layer">
  </div>
  <div class="parallax-layer position-relative" data-depth="0.2">
     <img src="path-to-image" alt="Layer">
  </div>
  <!-- Add as many layers as you need with different data-depth parameters for more 3D-like effect. Make sure add position-relative to biggest layer to keep width and height. -->
</div>
// Cursor position parallax
.parallax
  .parallax-layer.position-relative(data-depth="0.1")
    img(src="path-to-image", alt="Layer")
  .parallax-layer(data-depth="0.35")
    img(src="path-to-image", alt="Layer")
  .parallax-layer(data-depth="0.2")
    img(src="path-to-image", alt="Layer")
  //- Add as many layers as you need with different data-depth parameters for more 3D-like effect. Make sure add position-relative to biggest layer to keep width and height.
